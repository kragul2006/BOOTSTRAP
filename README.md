# 🌿 Gather — Community Event Portal

A modern, responsive community event discovery platform where moments become memories. Browse events, register with ease, and connect with local experiences.

## ✨ Features

- 🎨 **Editorial Design** — Warm terracotta & forest color palette with organic typography
- 📱 **Fully Responsive** — Works seamlessly on desktop, tablet, and mobile devices
- 🎯 **Category Filtering** — Filter events by Music, Tech, Food, Arts, Wellness & Literature
- 🔍 **Live Search** — Search events with real-time feedback
- 📝 **Registration Form** — Complete validation & success modal
- 🖼️ **Image Carousel** — Bootstrap-powered event gallery
- 📋 **Interactive Sidebar** — Price range filter, popular tags, organizer profile
- 🧩 **Accordion FAQ** — Collapsible support section
- 📧 **Newsletter Signup** — Multiple subscription points
- 🔼 **Back to Top Button** — Smooth scroll navigation
- 🍞 **Toast Notifications** — Custom feedback system

🚀 Quick Start Structure
To run this project locally:

bash
# Clone or download
git clone https://github.com/your-username/gather-event-portal.git

# Navigate to project
cd gather-event-portal

# Open in browser
open index.html    # Mac
start index.html   # Windows
No build tools or server required — it's pure HTML/CSS/JS!
